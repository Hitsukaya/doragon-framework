# Contributing

Thank you for considering contributing to Doragon.

## Ways to contribute

- Bug reports
- Feature suggestions
- Documentation improvements
- Pull requests

---

## Development Guidelines

- Keep code modular
- Follow existing CLI structure
- Prefer POSIX compatible shell patterns when possible
- Document new commands

---

## Pull Request Process

1. Fork repository
2. Create feature branch
3. Commit changes
4. Open pull request
